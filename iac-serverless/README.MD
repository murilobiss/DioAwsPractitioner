## 💎 Projeto

É uma API de listagem de raças de cachorros. Foi utilizado AWS lambda para esse projeto.

## 📦 Instalar

```sh
## instalar dependencias
❯ npm i

## rodar local
❯ npm run start

## deploy em ambiente de desenvolvimento AWS
❯ npm run deploy:dev

```

A Aplicação roda na porta **3000**.


## 🚀 Tecnologias

- nodejs
- javascript
- AWS Lambda
- serverless framework
